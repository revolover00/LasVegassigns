# Las Vegas Sign Company

A modern, highly performant web application custom-designed and engineered for **Las Vegas Sign Company**. Featuring architectural corporate showrooms, specialized commercial category selectors, responsive drag-and-drop structural estimators, and customized sizing dialogs.

Designed and developed by **Revo Code**.

---

## 🚀 Key Features

* **Custom Commercial Signage Portfolio**: Showcases real assemblies across 10 core distinct categories including Anodized Aluminum, Lobby Glass Directorys, Pylon Columns, and Window Decals.
* **Interactive Sizing Inspector**: Dynamic specs detailing structural US wind-loads, material grades (such as Marine Alloy H32), and zoning-compliance details.
* **Granular Architectural Estimator**: Robust file uploader supporting high-fidelity engineering vector blueprints (`.ai`, `.eps`, `.dxf`, `.pdf`, `.png`, `.jpg`).
* **Fluid Animation Profiles**: Immersive transitions and modals backed by custom `motion` presets for clean, distraction-free user flow.
* **Deployment Ready**: Fully configured with SPA routes fallback mechanisms optimized for immediate hosting on **Vercel** or traditional Node containers.

## 🛠️ Local Development Setup

Follow these simple guidelines to boot the environment locally on your terminal:

### 1. Prerequisites
Ensure you have **Node.js** (v18 or higher) installed on your system.

### 2. Installation
Clone the repository and install the production-grade dependencies:
```bash
npm install
```

### 3. Run Development Server
Spin up the local client application under hot-reloading configurations:
```bash
npm run dev
```
The server will boot up and expose the interface immediately.

### 4. Direct Production Build
To compile the static assets optimized with structural minification for secure storage:
```bash
npm run build
```

## 🌐 Deploy to Vercel

This repository has been configured with local rewrites inside `vercel.json` to handle React client-side routing securely.

To deploy instantly via Vercel CLI:
1. Install Vercel globally: `npm install -g vercel`
2. Run `vercel` from the root directory and link your account.
3. Done! Vercel will automatically parse standard SPA fallback scripts seamlessly without broken `/services` or `/gallery` routes.
